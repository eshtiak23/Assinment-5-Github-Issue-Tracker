1️⃣ What is the difference between var, let, and const?

var: It is the old way of declaring variables. It can be redeclared and updated.

let: It can be updated..

const: It is used to declare constant variables. It cannot be updated or redeclared after declaration.


2️⃣ What is the spread operator (...) ?

The Spread operator  is used to expand or spread elements of a array or object. It helps to copy, merge, or pass elements easily.


3️⃣ What is the difference between map(), filter(), and forEach() ?

map: It create a new array by applying a function to every element.

filter: It create a new array with elements that match a certain  condition.

forEach: It runs a function for each element but does not return a new array.


4️⃣ What is an arrow function?

 It is a shorter way to write functions in JavaScript, using =>.It makes the code simpler and cleaner.


5️⃣ What are template literals?

Template literals are used to create strings in JavaScript using backticks ( ).
